# 3. 技术架构

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **后端** | Flask + SQLAlchemy | Python Web框架 |
| **前端** | Bootstrap 5.3 | 现代化响应式UI |
| **数据库** | SQLite | 轻量级存储 |
| **缓存** | Redis | 高性能缓存 |
| **认证** | Flask-Login + Token | 双重认证机制 |

## 架构图

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   客户端     │────▶│  Flask API  │────▶│   Redis     │
│  (Web/API)  │     │   Server    │     │   Cache     │
└─────────────┘     └──────┬──────┘     └─────────────┘
                           │
                    ┌──────▼──────┐
                    │   SQLite    │
                    │  Database   │
                    └─────────────┘
                           │
         ┌─────────────────┼─────────────────┐
         ▼                 ▼                 ▼
    ┌─────────┐      ┌─────────┐      ┌─────────┐
    │  飞书   │      │  钉钉   │      │  Flomo  │
    └─────────┘      └─────────┘      └─────────┘
```
